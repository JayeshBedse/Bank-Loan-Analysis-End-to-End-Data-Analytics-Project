Bank Loan Analysis - README

This project presents a complete Bank Loan Analysis workflow built using SQL, Python, Excel, Power BI, and Tableau.
The goal is to analyse loan performance, track key lending metrics, and build interactive dashboards that help assess the health of the loan portfolio.

Tools Used

- SQL
- Python
- Excel
- Power BI
- Tableau

Key Features

- Data cleaning and KPI generation using SQL
- Exploratory Data Analysis (EDA) using Python
- Excel-based KPI reporting and interactive dashboard
- Power BI dashboards for high-level and detailed insights
- Tableau visualizations for interactive trend analysis

Key Metrics & KPIs

- Total Loan Applications
- Total Funded Amount
- Total Amount Received
- Average Interest Rate
- Average Debt-to-Income (DTI) Ratio
- MTD and MoM comparisons for all KPIs

Good vs Bad Loan KPIs


Good Loans: Fully Paid + Current

Bad Loans: Charged Off

Dashboards Included-

Dashboard 1: Summary – High-level KPIs and Good vs Bad Loan segmentation

Dashboard 2: Overview – Trends, regional analysis, loan purpose, home ownership insights

Dashboard 3: Details – Granular loan data including status, interest rate, DTI, and borrower information
